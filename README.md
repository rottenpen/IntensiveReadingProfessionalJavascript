# IntensiveReadingProfessionalJavascript
带着问题精读红宝书 第四版
## 前言
这个系列主要是为了带着大家有目的地读 JavaScript 高级程序设计（第4版）这本800多页的书。毕竟书本内容繁多，带着问题来读这本书，可以让大家更高效地阅读完这本书，也能补充阅读过程中会漏掉的重点细节。

持续更新中，觉得不错的可以点个 star 收藏支持。
